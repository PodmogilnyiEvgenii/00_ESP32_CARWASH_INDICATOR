Возможно потребуется установить поддержку Java, получить последнюю версию можно по ссылке: https://www.java.com/ru/download/manual.jsp

Удаление текущей и старых версий: https://www.java.com/ru/download/uninstalltool.jsp

Для упрощения запуска можно настроить асоциацию файлов *.jar c Java(TM) Platform SE binary
